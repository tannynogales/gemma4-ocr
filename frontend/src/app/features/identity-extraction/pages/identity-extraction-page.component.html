Qué hace realmente la POC
  • Hace una primera llamada sobre la imagen completa.
  • Si falta algo importante, puede hacer un recovery pass.
  • Si sigue faltando `documentNumber`, puede recortar esa zona y reintentar.
  • Después normaliza y guarda el snapshot exacto de esta configuración.

Si comparas con LM Studio manual, allá normalmente estás viendo sólo una llamada. La POC puede usar más de un paso.
